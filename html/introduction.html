Introduction

The ml-orchestrator project is designed to streamline the management of modular components in Kubeflow Pipelines (KFP), with a primary focus on Machine Learning (ML) workflows. For installation instructions and getting started, please refer to Usage Guide.

Concepts

The core concept of ml-orchestrator is to provide a framework designed for creating, managing, and maintaining Kubeflow components. It enables you to define component logic using dataclasses, which can easily be serialized into KFP strings. This structured approach simplifies component management, enhances maintainability, and facilitates dependency and configuration handling.
